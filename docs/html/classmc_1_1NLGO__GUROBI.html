Detailed Description

template<typename T>
class mc::NLGO_GUROBI< T >

mc::NLGO_GUROBI is a C++ class for global optimization of NLP and MINLP using complete search. Relaxations for the nonlinea or nonconvex participating terms are generated using MC++. Further details can be found at: Nonlinear Global Optimization using MC++ and GUROBI
